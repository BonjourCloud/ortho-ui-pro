## Adding New Images

1. Save your image to this folder
2. Use descriptive filenames (e.g., `acl-reconstruction-diagram.jpg`)
3. Recommended size: 1200px width (responsive)
4. Format: JPG or PNG
5. Optimize images before uploading (use tools like TinyPNG)

## Image Guidelines

- **Max file size:** 500KB (for fast loading)
- **Aspect ratio:** 16:9 or 4:3 preferred
- **Quality:** High resolution but optimized
- **Alt text:** Always include descriptive alt text for SEO and accessibility
