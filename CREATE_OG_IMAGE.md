# 🖼️ Create Open Graph Image for WhatsApp/Social Sharing

## What is an OG Image?

The Open Graph (OG) image is the preview image that appears when you share your website link on:
- WhatsApp
- Facebook
- Twitter
- LinkedIn
- Other social platforms

## ✅ What I Updated

I've updated the meta tags to point to `https://orthocarehub.in/og-image.jpg` instead of the Lovable image.

## 🎨 Create Your OG Image

### Required Specifications:
- **Size:** 1200 x 630 pixels (required by Facebook/WhatsApp)
- **Format:** JPG or PNG
- **File size:** Under 8MB (preferably under 1MB)
- **File name:** `og-image.jpg`
- **Location:** `public/og-image.jpg`

### Design Recommendations:

**Include:**
- ✅ Clinic name: "Dr. Srivanth Dasari Ortho Clinic"
- ✅ Tagline: "Premium Orthopedic Care in Hyderabad"
- ✅ Key services: "Joint Replacement | Sports Medicine | Trauma Care"
- ✅ Contact: Phone number or website
- ✅ Professional photo (optional)
- ✅ Brand colors (teal/green theme)

**Avoid:**
- ❌ Too much text (keep it readable on mobile)
- ❌ Small fonts (minimum 40px for main text)
- ❌ Important content near edges (safe zone: 100px margin)

## 🔄 Upload Your OG Image

### Step 1: Create the Image
- Use one of the options above
- Make sure it's 1200 x 630 pixels
- Save as `og-image.jpg`

### Step 2: Add to Your Project

**If deploying via Vercel:**
1. Add `og-image.jpg` to your `public/` folder
2. Commit and push to GitHub:
   ```bash
   git add public/og-image.jpg
   git commit -m "Add Open Graph image"
   git push
   ```
3. Vercel will auto-deploy

**If using FTP/cPanel:**
1. Upload `og-image.jpg` to your website's root directory
2. Make sure it's accessible at: `https://orthocarehub.in/og-image.jpg`

### Step 3: Verify Upload

Visit: https://orthocarehub.in/og-image.jpg

You should see your image. If you get a 404, the file isn't uploaded correctly.

## 🧪 Test Your OG Image

### Test on WhatsApp:

1. **Clear WhatsApp cache:**
   - WhatsApp caches link previews
   - You may need to wait 24 hours for cache to clear
   - Or use a different link format (add ?v=2 at the end)

2. **Test the link:**
   - Share: `https://orthocarehub.in?v=2`
   - WhatsApp should fetch new preview

### Test with Facebook Debugger:

1. **Go to:** https://developers.facebook.com/tools/debug/
2. **Enter:** https://orthocarehub.in
3. **Click:** "Scrape Again" button
4. **Check:** Preview shows your new image

### Test with LinkedIn:

1. **Go to:** https://www.linkedin.com/post-inspector/
2. **Enter:** https://orthocarehub.in
3. **Check:** Preview

## 🆘 Troubleshooting

### WhatsApp still shows old image:
- WhatsApp caches for 24 hours
- Use Facebook Debugger to force refresh
- Share with `?v=2` parameter: `https://orthocarehub.in?v=2`

### Image not showing:
- Check file is at: `https://orthocarehub.in/og-image.jpg`
- Check file size is under 8MB
- Check dimensions are exactly 1200x630

### Image looks blurry:
- Make sure you're exporting at full resolution
- Don't scale up a smaller image
- Create at 1200x630 from the start
